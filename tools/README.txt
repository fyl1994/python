目前只支持Ubuntu环境下的xlsx格式的Excel

1.切换到MTK_NV_Daily目录下,下载代码:
    git clone https://github.com/fyl1994/python.git

2.安装python 3;
    sudo apt install python3.4

3.使用:
3.1.将脚本tools文件夹放在MTK_NV_Daily同级目录下.
3.2.修改脚本odmnvexcel.sh的权限
    sudo chmod 777 odmnvexcel.sh
3.3.执行odmnvexcel.sh脚本,根据提示输入即可

NOTE:
    源代码可以根据自己的需求,先fork到自己的GitHub上之后,再自行修改
